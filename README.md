# `Lab 2 E-Commerce `

## Project Setup

```
npm install
```

### Runs the app in the development mode on [http://localhost:3000]

```
npm start
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
